# IVD Private / Planning Documentation

> **NOTE**: This directory is committed during the private alpha phase to enable
> multi-device work on the book and community strategy. It will be **removed
> before public launch** (Phase 3 — see `community/community_adoption_intent.yaml`).

This directory contains private planning artifacts for the IVD launch strategy.
It is **not** part of the core IVD framework and is skipped by embedding tools
(prefix `_`).

## Contents

### `community/`

Private adoption and community launch planning.

- **community_adoption_intent.yaml** — Intent artifact for IVD community adoption
  and standalone launch strategy (phases, constraints, risks, public launch checklist)
- **migration_plan.md** — Historical document: how IVD was extracted into a
  standalone repository (completed, kept for reference)

## Pre-Public-Launch Checklist

Before making the repo public (Phase 3), this directory and the `book/` directory
must be removed, and embeddings regenerated without book content. Full details in
`community/community_adoption_intent.yaml` under the `private_content_removed_before_public`
constraint and Phase 3 description.

**Note:** This directory starts with `_` to indicate it's not part of the core
distributable framework.

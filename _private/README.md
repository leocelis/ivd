# IVD Private / Historical Documentation

⚠️ **NOTE**: This directory is committed during the private alpha phase to enable multi-device work on the book and community strategy. It will be **removed before public launch** (Phase 3 - community adoption).

This directory contains historical evolution documents and private planning artifacts. It is **not** part of the core IVD framework and is skipped by embedding tools (prefix `_`).

## Contents

### `ai_driven_dev/`

Historical documentation showing how IVD emerged from real AI-driven development sessions.

- **from_lp_to_ivd.md** - Evolution from Literate Programming to Intent-Verified Development
- **ai_driven_development_guide.md** - Extracted insights from training sessions (10,500+ lines)
- **meetings_index.md** - Processing tracker for 18 training sessions
- **meetings.md** - Raw transcripts from training sessions (1,283 lines)
- **README.md** - Overview of the historical documentation

### `community/`

Private adoption and community launch planning.

- **community_adoption_intent.yaml** - Intent artifact for IVD community adoption and standalone launch strategy

## Purpose

These documents are preserved for:

- Understanding the "why" behind IVD principles (historical context)
- Tracing the evolution from Literate Programming to IVD
- Private planning for community adoption and standalone launch
- Examples of problems IVD solves (from real training sessions)

**Note:** This directory starts with `_` to indicate it's not part of the core distributable framework.

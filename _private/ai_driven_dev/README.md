# IVD Historical Documentation

This directory contains the historical evolution of IVD, preserved for context and understanding.

## Contents

- **from_lp_to_ivd.md** - Evolution from Literate Programming to Intent-Verified Development
- **ai_driven_development_guide.md** - Extracted insights from training sessions (8,870 lines)
- **meetings_index.md** - Processing tracker for 18 training sessions
- **meetings.md** - Raw transcripts from training sessions (1,283 lines)

## Purpose

These documents show how IVD emerged from real development sessions. They're preserved for:

- Understanding the "why" behind IVD principles
- Historical context for framework decisions
- Examples of problems IVD solves

Note: This directory starts with `_` to indicate it's not part of the core framework and is skipped by embedding tools.
